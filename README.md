## Контрольная Вариант 20

**_Тема:_** Создание абстрактного класса `State` и его производных с реализацией интерфейсов `IComparable`, `ICloneable`.  
**_Язык программирования:_** C#  
**_Среда разработки:_** Visual Studio / VS Code

---

### Задача:

1. Создать **абстрактный класс `State`**, реализующий функционал вывода информации о государстве и вычисления его возраста.
2. Реализовать в классе интерфейс `IComparable` для возможности сортировки по **численности населения**, а также обеспечить сортировку по:
    - дате основания
    - имени руководителя
3. Создать производные классы:
    - `Republic`
    - `Kingdom`
    - `Federation`
4. Каждый из классов должен:
    - Наследовать поля от базового класса
    - Иметь собственные уникальные поля
    - Реализовывать интерфейс `ICloneable`
5. Создать обобщённый список (`List<State>`) и:
    - Вывести всю информацию
    - Отсортировать по численности населения
    - Найти государства, численность населения которых попадает в заданный диапазон

### Задание 2

Реализовать класс хеш-таблиц `Hash` для хранения элементов описанной в задании 1 иерархии классов.

```csharp
class Hash : IEnumerable
{
    protected ArrayList keys;
    protected List<State> values;
    ...
}
```

**Требования:**

1. **Конструкторы:**

    - `public Hash()` - создание пустой коллекции
    - `public Hash(int capacity)` - создание пустой коллекции с начальной ёмкостью
    - `public Hash(Hash h)` - создание коллекции, инициализированной элементами и ёмкостью из параметра

2. **Реализовать:**

    - a) Свойство `Count` - количество элементов в коллекции
    - b) Индексатор для доступа по ключу
    - c) Методы добавления: `Add(object key, State t)` и `AddRange(Hash h)`
    - d) Методы удаления: `Remove(object key)` и `RemoveAll(Hash h)`
    - e) Метод поиска: `Find(State value)`
    - f) Метод клонирования коллекции
    - g) Метод поверхностного копирования
    - h) Метод очистки коллекции
    - i) Интерфейсы `IEnumerable` и `IEnumerator` (реализация итератора)

3. **В методе `Main()`:**
    - a) Заполнить коллекцию объектами. Используя меню, реализовать добавление и удаление объектов
    - b) Разработать и реализовать три запроса (количество элементов определённого вида, печать элементов определённого вида и т.п.)
    - c) Выполнить перебор элементов коллекции с помощью `foreach`
    - d) Выполнить клонирование коллекции
    - e) Выполнить поиск заданного элемента в коллекции
